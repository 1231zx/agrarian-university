Документы для поступления

Перечень документов для поступления

Основные документы

  • Заявление о приеме
  • Документ, удостоверяющий личность, гражданство
  • Документ установленного образца об образовании (аттестат/диплом)
  • Документ, подтверждающий регистрацию в системе индивидуального (персонифицированного) учета (СНИЛС)
  • Фотографии поступающего (4 шт.)
  • Медицинская справка (для отдельных направлений подготовки)

Документы для лиц с особыми правами

  • Документы, подтверждающие инвалидность
  • Документы, подтверждающие статус сироты
  • Документы победителей олимпиад
  • Договор о целевом обучении

Документы, подтверждающие индивидуальные достижения

  • Дипломы победителей олимпиад
  • Сертификаты участников конференций
  • Грамоты и благодарственные письма

Примечание: При подаче документов через операторов почтовой связи, необходимо приложить копии документов, заверенные в установленном порядке.
